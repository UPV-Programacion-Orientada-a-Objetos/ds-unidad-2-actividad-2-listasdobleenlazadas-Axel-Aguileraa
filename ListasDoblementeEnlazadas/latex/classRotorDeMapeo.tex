\doxysubsection{Detailed Description}
Implementa una lista circular doblemente enlazada para el cifrado. 

Actúa como un "{}disco de cifrado"{} que mantiene el alfabeto (A-\/Z) y puede rotar para cambiar el mapeo de caracteres.

Calcula el carácter decodificado basado en la rotación actual. 

Esta es la lógica central. Toma un carácter de entrada (ej. \textquotesingle{}W\textquotesingle{}) y calcula su offset desde \textquotesingle{}A\textquotesingle{} (ej. 22). Luego, se posiciona en la \textquotesingle{}cabeza\textquotesingle{} actual (ej. \textquotesingle{}C\textquotesingle{}) y avanza ese mismo offset (22 pasos) para encontrar el carácter mapeado (ej. \textquotesingle{}Y\textquotesingle{}).
